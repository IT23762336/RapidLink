# RapidLink

RapidLink is a local-first distributed messaging system prototype built in Python.

## Quick Start
1. Install dependencies from `docs/requirements.txt`.
2. Run `python3 scripts/dm_gateway.py`.
3. Open `http://localhost:8081/`.

## Useful Commands
- `python3 scripts/run_node.py --id n1`
- `python3 scripts/run_cluster.py`
- `python3 scripts/failover_demo.py`

## Notes
- screenshots live under `public/`
- cluster layout lives in `config/cluster.yaml`
